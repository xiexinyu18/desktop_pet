# 桌宠 - 健康管理与生命周期版

**版本：1.0.0**（第一版，后续开发均基于此版本）

## 产品定位

陪伴型桌面宠物，支持与真实宠物联动、健康管理与生命周期。桌面宠物常驻电脑桌面，提供喂食/梳毛/猫砂提醒、宠物档案、摄像头检测联动等能力。

## 功能模块（1.0.0 MVP）

- **桌面端**：宠物窗口常驻、拖拽、基础动效
- **权限**：主人 / 游客身份，游客仅虚拟互动
- **宠物档案**：虚拟生日、基础档案存储
- **健康提醒**：喂食、梳毛、猫砂清理提醒
- **摄像头联动**：检测到宠物时“眼睛一亮”等动效（可选依赖摄像头）

## 环境要求

- Python 3.8+
- 桌面环境（Windows / macOS / Linux）

## 安装与运行

```bash
cd desktop_pet
pip install -r requirements.txt
python -m desktop_pet.main
```

或安装为包后运行：

```bash
pip install -e .
desktop-pet
```
